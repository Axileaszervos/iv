# ΠΑΡΑΔΟΤΕΟ 1 - ΕΙΣΑΓΩΓΗ
Στο μάθημα οπτικοποίηση της πληροφορίας, ευελπιστώ να μάθω μα χρησιμοποιώ νέα εργαλεία και ταυτόχρονος να εξοικειωθώ ακόμη περισσότερο με τα ήδη υπάρχοντα που έχω μάθει. Στόχος μου αποτελεί, η μάθηση νέων γνώσεων και η εφαρμογή τους σε πρακτικές καταστάσεις, όπως η ανάλυση δεδομένων και η δημιουργία window manager μέσω του arch linux. Τέλος, η ανάπτυξη ικανοτήτων συνεργασίας και επικοινωνίας αποτελεί ακόμη ένα σημαντικό στόχο για εμένα.
